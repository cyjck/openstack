## FTP安装与使用

#### 安装vsftpd服务[xserver1]

```
yum install vsftpd net-tools -y
```
<!-- more -->
#### 编辑配置文件

```
vi /etc/vsftpd/vsftpd.conf
```

```
anon_root=/opt
```

#### 启动服务

```
systemctl start vsftpd; systemctl enable vstpd
```

#### 提交查询结果

```
netstat -ntpl
```


#### 配置FTP客户端[xserver2]

```
vi /etc/yum.repos.d/ftp.repo
```

```
[ftp]
name=ftp
baseurl=ftp://192.168.100.20/centos
gpgcheck=0
enabled=1
```
